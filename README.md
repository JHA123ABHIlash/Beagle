# Beagle Blog 🐶

A simple static webpage about the **Beagle** dog breed, built with HTML & CSS.

## 📄 About

This page contains information about the Beagle breed including:
- Breed description
- Images
- Trivia (height, weight, coat, lifespan)

## 🛠️ Tech Stack

- HTML5
- CSS3
